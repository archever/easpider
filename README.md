# easpider 易用的爬虫框架

简单易用的爬虫框架
* 实现 redis 分布式, 或者本地队列爬取
* gevent 协程并发
* 自定义异常处理

## 使用说明
### 配置
`settings.py` 配置文件
```python
# 每次请求的延迟时间
SLEEP = 0
TIME_OUT = 10
# redis 配置
REDIS_CONF = {
    'host': 'localhost',
    'port': 6379,
    'db': 0,
}
# 后台 队列, 本地用 queue, 分布用 redis
# BACKEND = 'queue'
BACKEND = 'redis'
```

后台队列支持 redis 分布式, 和本地的 queue
